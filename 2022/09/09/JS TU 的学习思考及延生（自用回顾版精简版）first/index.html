7.prototypal inheritance model:

javascript uses prototypal inheritance model. That means that every constructor function/class has prototyppe property that is shared by every instance of the constructor/class. So any properties and methods or prototype can be access by every instance. prototype property returns a object

原型继承模型：javascript使用原型继承模型。这意味着每个构造函数函数/类都有prototyppe属性，该属性由构造函数/类的每个实例共享。因此，每个实例都可以访问任何属性、方法或原型。prototype属性返回一个对象
